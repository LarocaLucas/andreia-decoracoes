## Sobre o Site

### Design
- Estética elegante/luxo — fundo escuro quente, dourado e off-white
- Fontes: **Cormorant Garamond** (display) + **Great Vibes** (script) + **Jost** (corpo)
- Hero com zoom suave e parallax
- Foto do sobre com efeito parallax no scroll
- Galeria por álbuns com rotação automática e navegação por categoria

### Estrutura
```
/
├── index.html
├── css/
│   └── style.css
├── js/
│   └── main.js
└── assets/
    └── images/
        ├── logo.png
        ├── foto-hero.jpg
        └── galeria/
            └── (fotos por categoria)
```

### Funcionalidades
- Navegação fixa com scroll suave
- Cursor customizado
- Animações de entrada (reveal on scroll)
- Galeria por álbuns — Casamentos, Infantil, 15 Anos, Corporativo
- Rotação automática de fotos com autoplay (5s)
- Navegação por setas e dots na galeria
- Formulário de contato integrado ao WhatsApp
- Botão flutuante WhatsApp
- Responsivo para mobile
